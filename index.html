Conectar a um amigo:
Para conectar-se um amigo, você deve solicitar que o mesmo entre no site e lhe forneca o ID. Com esse ID, você poderá se conectar ao amigo via chamada de video conferencia.

Esperar por conexão
Forneça o ID gerado na caixa "Esperar por conexão" ao seu amigo para que o mesmo possa conectar se a você. Se peferir, forneca o link ao seu amigo para facilitar na conexão.
